JNI 실습을 진행하였다.

<H2>자바 JNI 코드 작성</H2>
<pre>
<code>public class JNI {
	static {
        System.loadLibrary("JNI");
    }

    private native int getNumber();

    private native void printHelloWorld();

    public static void main(String[] args) {
        JNI jni = new JNI();

        jni.printHelloWorld();
        System.out.println(jni.getNumber());
    }
}
</code>
</pre>
